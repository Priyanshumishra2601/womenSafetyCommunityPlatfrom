# 🚨 Women Safety Community Platform

A real-time women safety platform designed to provide instant emergency assistance using community support and location-based alert systems.

## 🌟 Features

- 🚨 One-tap SOS Emergency Alert
- 📍 Live Location Tracking (Real-Time)
- 👥 Community-Based Alert System (Nearby Users)
- 🚓 Nearest Police Station Notification
- 📲 Emergency Contact Alerts
- 🔔 Instant Notifications
- 🧭 Safe Route Suggestions (Planned)
- 🔐 Secure Authentication System

---

## 🧠 How It Works

1. User logs in and enables location access  
2. In an emergency, user presses the SOS button  
3. System instantly:
   - Shares live location  
   - Alerts nearby users  
   - Notifies nearest police station  
4. Nearby users can respond and assist in real-time  

---

## 🛠️ Tech Stack

- **Frontend:** React.js / Next.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Realtime:** Socket.io / Firebase  
- **APIs:** Google Maps API
